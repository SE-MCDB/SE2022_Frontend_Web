## Description

This is a Web Frontend code of the `PaperDaily` project based on [`Vue.js`](https://vuejs.org)

## Quickstart

1. Install dependencies

```bash
$ npm install
```

2. Run

```
$ npm run dev
```

3. Build

```bash
$ npm run build
```
